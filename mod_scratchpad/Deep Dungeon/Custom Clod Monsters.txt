== Clod Monsters ==
// Shoots pellets at the player
- attack_legion: Number of pellets
- attack_radial_degree: Attack cone size/shape
  - For ironclod, it has a value of 45 which shoots 3 pellets in a 45-degree cone, despite not having attack_legion set.
- projectile_speed: As named
- attack_sequence: How many attacks in a row
- attack_sequence_turn: how far to turn between attacks, uses attack_sequence
- attack_sequence_image_speed: ???

// Mutually exclusive with above parameters
- split_depth: how many pellets a single pellet splits into
- split_angle: angle of the split pellets
- split_distance: distance until split
- split_depreciation: ???
